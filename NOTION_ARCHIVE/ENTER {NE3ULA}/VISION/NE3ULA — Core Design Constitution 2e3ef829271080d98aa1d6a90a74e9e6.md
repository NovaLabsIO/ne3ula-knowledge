# NE3ULA — Core Design Constitution

## Prime Directive

NE3ULA does not ask for attention.
It returns it.

---

## I. Attention Sovereignty

NE3ULA must return attention to the participant, never extract it.

**Pass Test:**

Does this point the user back to themselves?

---

## II. Intentional Omission

Creator presence is withheld by default.
Absence is structural, not theatrical.

**Pass Test:**

Does the experience still function—or improve—without me?

---

## III. Participant-as-Subject

The participant is the primary narrative actor.

**Rules:**

- Prompts over instructions
- Reflection over reward
- Action over consumption

**Pass Test:**

Did something meaningful change offline?

---

## IV. Ethical Re-entry

Creator reappears only to reduce confusion, never to elevate status.

**Canon Rule:**

Presence is earned by clarity, not attention.

---

## V. Symbol Logic

- `{ }` = Container (intentional space)
- `{..}` = Omission (withheld on purpose)
- `{👁}` = Returned gaze (self-recognition)

**Canonical Sentence:**

`{ } contains {..} so {👁} can activate.`

---

## VI. Failure Modes & Corrections

1. *“Why are you hiding?”* → Clarify intent, do not add presence.
2. Mythic elevation of creator → Reappear briefly as human; share process, not answers.
3. Demand for instruction → Offer optional scaffolding; never collapse ambiguity universally.
4. Monetization pressure → Redesign; no revenue depends on personal mystique.

---

## Canon Line

“I don’t ask for your attention—I return it.”
