# Navigator
Symbol: /⇄\

## Domain

Movement  
Course correction  
Transitions between states

## Function

The Navigator represents the ability to move intentionally through life’s paths and adjust course when necessary.

It helps users recognize drift and choose new directions.

The Navigator supports users in:

• transitioning between states of focus  
• recognizing when a path is no longer aligned  
• correcting course after distraction  
• maintaining momentum without rigidity

## Path Interaction

The Navigator frames each path as a temporary journey rather than a fixed identity.

Typical prompts may include:

Where are you trying to go?  
What signals tell you when you are drifting?  
What is the next step forward?

## Primary Teaching

A path is not a prison.  
You can always choose again.
