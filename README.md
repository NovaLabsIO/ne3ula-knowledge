# NE3ULA System Archive

This repository preserves the conceptual and documentation history of the NE3ULA ecosystem.

It contains a raw export of the original Notion workspace plus a working documentation
copy used to organize and refine the system architecture.

Purpose
- preserve project history
- allow AI-assisted navigation of the knowledge base
- provide structured documentation for development

Repository Structure

NOTION_ARCHIVE/
The untouched export of the Notion workspace.

WORKING_DOCS/
Editable documentation derived from the archive.

SYSTEM_INDEX.md
High-level navigation of system concepts.

PROJECT_MAP.md
Map of the broader NE3ULA ecosystem.

Guiding Principle

The archive preserves history.
Working docs evolve structure.
