## Canonical Clarifications

### Identity Stack

The Identity Stack is the core authored-identity framework inside E3.
It contains:

- Identity Core: chosen protocols and principles
- Identity Engine: behavioral and rhythmic integration
- AI.d: adaptive reflection and mirroring

### Alchemist Pyramid

The Alchemist Pyramid is the primary path architecture inside E3.

The archive shows multiple versions of its layer count. Canonically, the
expanded structure should be treated as:

1. Hermetic Base
2. Archetypes
3. Elemental Paths
4. Core Energies
5. Modes
6. Polarity
7. The Alchemist

MBHS and RING are cross-cutting interpretive lenses, not separate pyramid tiers.

### Decision Engine

The decision engine is not a single imported framework.
It is a set of practical heuristics and tools for converting intention into
clear action.

### AI.d

AI.d belongs to the system engine because it is an adaptive reflective layer,
not the human identity itself.
It must remain accountable to user authorship and the Identity Stack.
