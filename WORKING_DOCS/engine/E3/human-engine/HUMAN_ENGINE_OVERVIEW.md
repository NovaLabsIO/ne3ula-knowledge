# Human Engine Overview

The human engine is the participant-development layer of E3.

It contains the frameworks that help a person:

- author identity
- interpret experience
- move through transformation
- build aligned rhythms
- make more conscious decisions

## Core Subsystems

### Identity Stack

The Identity Stack is the authored identity framework of E3.
It defines:

- what the participant stands for
- how those values are translated into behavior
- how reflection and adaptive mirroring support recalibration

### Alchemist Pyramid

The Alchemist Pyramid is the primary path architecture of E3.
It organizes transformation as a layered and cyclical process rather than a
simple ladder of advancement.

### Decision Engine

The decision engine collects practical heuristics for prioritization, action,
habit formation, and self-alignment.

## Cross-Cutting Lenses

Two cross-cutting interpretive systems recur throughout the archive and should
be treated as human-engine constants:

### MBHS

Mind, Body, Heart, Spirit provides a four-part human integration lens.
It helps reflection include cognition, embodiment, emotion, and meaning rather
than collapsing everything into thought alone.

### RING

Mythos, Logos, Pathos, Ethos provides a four-angle reflection structure for
understanding story, logic, emotion, and values within any path or event.

## Operational Pattern

The human engine is designed to move insight into embodiment.

Canonical movement:

signal
-> reflection
-> interpretation
-> choice
-> rhythm
-> recalibration
-> return

The point is not perfect self-management.
The point is conscious becoming.
