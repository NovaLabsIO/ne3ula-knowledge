# Polarity

Polarity is the reflective integration layer of the Alchemist Pyramid.

It is not a path of external tasks. It is the layer where contradiction,
tension, and opposition are held consciously enough to become transformative.

## Canonical Function

Polarity exists to:

- expose false binaries
- reveal tension fields inside identity
- challenge premature certainty
- prepare the participant for deeper integration
