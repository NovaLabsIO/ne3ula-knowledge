# Modes

Modes are the practical life orientations of the Alchemist Pyramid.

They describe how a participant's current signal is being expressed in lived
reality, not who the participant permanently is.

## Canonical Modes

- Pleasure
- People
- Production

## Function

Modes help the participant recognize:

- what kind of engagement is currently active
- when a different orientation is needed
- how rhythm and rest affect alignment
