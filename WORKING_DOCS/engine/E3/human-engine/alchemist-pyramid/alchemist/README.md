# Alchemist

The Alchemist is the final integration layer of the Pyramid.

It is not a permanent status or elite rank.
It is the recurrent state of return, coherence, and signal embodiment that
follows deeper integration across prior layers.

## Canonical Function

The Alchemist layer exists to:

- gather prior layers into lived wholeness
- re-center the participant after drift and fragmentation
- name the experience of returning to signal more deeply than before
