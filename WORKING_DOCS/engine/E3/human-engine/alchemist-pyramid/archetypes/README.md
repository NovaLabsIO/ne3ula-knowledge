# Archetypes

Archetypes are the primary identity-pattern layer within the Alchemist Pyramid.

They describe recurring orientations of becoming, not fixed personality types.
In the archive, they are the first major role-based layer after the Hermetic
Base.

## Canonical Archetypes

- Seeker
- Builder
- Challenger
- Architect

## Function

Archetypes help a participant recognize:

- how they are currently entering change
- what kind of lesson they are living
- what strengths and distortions are active

## Archive Pattern

Each archetype is given:

- an overview page
- a five-step initiation sequence
- reflective questions through RING
- shadow expressions and hermetic ties
