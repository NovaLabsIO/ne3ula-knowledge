# Mirror

Mirror is the path of self-recognition, contradiction, and honest reflection.

## Function

Mirror serves as an entry path into transformation by asking the participant to
see themselves beyond roles, distortions, and inherited masks.

## What It Works On

- self-perception
- contradiction
- hidden emotional truth
- misalignment between values and behavior

## Step Progression

1. `Facing the Reflection`
   Name the story, patterns, and values of the current self.
2. `The Fragmented Image`
   Notice contradiction, masking, and distortion.
3. `The Echo Beneath`
   Listen for buried truth and quieter signal.
4. `Through the Cracks`
   Integrate the broken or disowned parts.
5. `The Clean Reflection`
   Reclaim a more truthful orientation toward self.

## Canonical Role

Mirror is the foundational reflective path for identity clarification.
