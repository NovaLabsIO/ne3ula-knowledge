# Elements

The Elements layer contains the operational practice paths of the Alchemist
Pyramid.

These are the terrains a participant actually walks in order to reflect,
integrate, and recalibrate.

## Canonical Elements

- Mirror
- Shadow
- Flame
- Frequency
- Vision

## Canonical Function

Elements are distinct from Core Energies.

- Core Energies describe underlying drivers
- Elements describe the operational path of practice

Each element in the archive includes:

- a path overview
- five RING-driven steps
- optional AI.d sync logic
- a completion state that redirects into the larger Pyramid
