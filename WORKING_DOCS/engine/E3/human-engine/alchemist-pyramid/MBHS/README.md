# MBHS

MBHS stands for:

- Mind
- Body
- Heart
- Spirit

It is a human-integration lens used across the Alchemist Pyramid and the wider
human engine.

## Function

MBHS exists to prevent transformation work from collapsing into only one kind of
intelligence.

It keeps reflection distributed across:

- cognition and interpretation
- embodiment and physical state
- emotion and relational tone
- meaning, paradox, and deeper orientation

## Canonical Use

MBHS should be treated as a cross-cutting lens rather than a separate pyramid
tier.

It helps ask:

- What is happening in the mind?
- What is happening in the body?
- What is happening in the heart?
- What is happening in the spirit?

## Archive Condition

The archive names Physical, Emotion, Soul, and Logic engines but leaves most of
those pages structurally undeveloped.

The durable concept is not a finished subframework for each page.
It is the four-part interpretive lens itself.
