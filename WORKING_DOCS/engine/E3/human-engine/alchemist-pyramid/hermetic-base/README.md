# Hermetic Base

The Hermetic Base is the foundational layer of the Alchemist Pyramid.

It provides the interpretive laws or principles that frame the rest of the
system. In the archive, this layer is explicitly described as the ground beneath
all other paths.

## Canonical Function

The Hermetic Base exists to:

- give structural language to reality, rhythm, and change
- frame transformation without becoming rigid doctrine
- provide recurring reference points for later Pyramid layers

## Canonical Principles

The archive names seven principles:

- Mentalism
- Correspondence
- Vibration
- Polarity
- Rhythm
- Cause and Effect
- Gender

These are treated as mirrors of reality rather than commandments.
