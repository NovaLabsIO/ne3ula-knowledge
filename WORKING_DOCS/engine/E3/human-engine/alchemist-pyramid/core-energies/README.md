# Core Energies

Core Energies are the fundamental forces named explicitly by the archive as:

- Freedom
- Thought
- Purpose
- Love

## Canonical Function

These are not the same as Elements.

Core Energies are the underlying motivational or existential forces behind why a
participant moves.
Elements are the operational paths through which those deeper forces become
observable and workable.

## Canonical Relationship

Core Energies sit between Archetypes and Elements in the reconstructed Pyramid
because they describe the deeper fuel behind the path, while Elements describe
the terrain of practice.
