# Eisenhower Matrix

The Eisenhower Matrix is one decision-support tool within the E3 decision
engine.

## Role in E3

Its function is simple:

- separate urgency from importance
- reduce reactive behavior
- preserve attention for aligned action

Within E3, this tool should be understood as a practical prioritization lens,
not as the full philosophy of decision-making.

It is most useful when combined with:

- Identity Core commitments
- current Mode awareness
- friction mapping
- the One Thing focus question

That combination keeps prioritization tied to signal rather than generic
productivity pressure.
