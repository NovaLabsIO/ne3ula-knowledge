# NE3ULA Knowledge Map

This document describes the conceptual architecture of the NE3ULA system.

It explains how the major layers of the project relate to each other.

This map is distinct from PROJECT_MAP.md, which describes repository structure.

---

# Core Structure

NE3ULA consists of several interacting layers.

NE3ULA is the world.
E3 is the engine.
AXZIO is the interface.

Domains express the world.
Systems enable participation.

# Site Pillars

The public presentation of the system aligns with four pillars.

MYTH
→ The meaning and narrative of the world.

E3
→ The transformation engine.

AXZIO
→ The interface layer.

MUSIC
→ The expressive domain that often acts as the entry signal.

---

# Summary

NE3ULA is the world.

E3 is the engine that transforms people inside the world.

AXZIO is the interface that allows users to navigate the system.

Domains express the outcomes of transformation.

Systems enable participation.

The goal of the system is to help participants live their legend.
